# BuscadorCIRepos
Buscador de repositorios CI en GitHub y GitLab

## Set Up 📋

Pre-requisites to reproduce the work:

- Python 3.9
- Git 2.32+

Install Python libraries

```
$ pip install -r requirements.txt
```
